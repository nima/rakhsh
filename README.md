# Rakhsh `rx`

A Neovim-based IDE stripped to essentials: fast, stable, predictable.  No bloat, with a small set of powerful features.

> …and there he stood — the great horse — his coat the color of dawnfire, waiting as though he had known all along that the path would bend this way.  Some said he chose the hero before the hero ever chose him; others whispered that he was born of wind and red earth, stubborn as mountain stone, loyal beyond reason.  Wherever the wanderer went, the great beast followed — not behind, but beside — and when the hills trembled and the rivers ran backward, it was he who carried the path forward, as though the world itself yielded to their stride…
>
> In the old chronicles he is remembered by a single name: **Rakhsh**.

## Core Goals

1. LSP diagnostics (file and project level)
2. Reliable symbol navigation (LSP + tags)
3. Fast project search (fuzzy + regex)
4. Quick file switching (MRU‑first)
5. Ad‑hoc regex highlighting
6. Nothing else unless it earns its place

## Integrations & Dependencies

- Neovim
- iTerm2

## Install

```bash
git clone https://github.com/nima/rakhsh
cd rakhsh
make install
rehash
rx <file>
```
